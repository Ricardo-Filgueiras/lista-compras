# Fluxo do Projeto: Lista de Compras Colaborativa para Papelaria

Este documento detalha o fluxo de funcionamento e os requisitos do sistema de listas de compras, focado em agilizar o atendimento e permitir a personalização pelo cliente.

## 1. Distribuição do Template (Link Original)
O processo inicia quando a papelaria disponibiliza um **link de template** (uma lista pré-preenchida com itens sugeridos). 
* Ao clicar no link, o sistema gera automaticamente uma **cópia exclusiva** dessa lista para o cliente.
* A lista original permanece intacta, servindo apenas como modelo para novos usuários.

## 2. Personalização e Edição do Cliente
Com sua própria cópia da lista, o cliente tem autonomia total para gerenciar os itens:
* **Adição e Remoção:** O cliente pode retirar itens que não deseja ou adicionar novos produtos do catálogo.
* **Substituição:** Caso um item não seja exatamente o que ele procura, ele pode substituí-lo por outro de sua preferência.
* **Ajuste de Quantidades:** Definir exatamente quanto de cada item será necessário para o seu pedido.

## 3. Colaboração via Link Compartilhável
O sistema permite que o cliente convide outras pessoas para editarem a mesma lista em conjunto:
* O cliente gera um **link de acesso** para enviar a outros usuários (familiares, amigos ou sócios).
* Todos os usuários com esse link podem visualizar e editar os itens simultaneamente antes do fechamento.

## 4. Finalização e Travamento (Checkout)
Assim que a lista estiver pronta, o cliente a marca como **"Concluída"**. 
* **Regra de Negócio:** Após a conclusão, o cliente não pode mais alterar os itens, garantindo que o pedido enviado seja o mesmo que será separado.
* A lista é enviada automaticamente para o painel administrativo da papelaria.

## 5. Separação Logística e Retirada
Com a lista final em mãos, a equipe da papelaria inicia o processo interno:
* **Separação:** Um funcionário localiza e separa cada item físico no estoque.
* **Pronto para Retirada:** Quando o cliente chega à papelaria, o pe
ido já está conferido e embalado.
* **Objetivo:** Eliminar o tempo de espera do cliente dentro da loja e otimizar o fluxo de trabalho dos funcionários.
